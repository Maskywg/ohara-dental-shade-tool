# 大原牙醫瓷牙比色工具

以瀏覽器本機處理照片的瓷牙比色工具原型。

工具目的：協助牙科技師確認完成瓷牙的色系是否與診所臨床自然牙目標色彩偏離過多，並以校正後 Lab 與 ΔE00 作為修色及溝通依據。

## 目前功能

- 臨床牙與照片內比色板雙取樣
- 完成瓷牙與照片內比色板雙取樣
- 兩張照片各自獨立校正
- 校正後 CIE Lab 數值
- CIEDE2000 色差比較
- 近似 VITA Classical 色階
- Google Docs 相容報告匯出
- 診所建立案件、技工所讀取、色差判定與結案流程
- 診所可下載內含臨床照片與校正資料的案件 JSON，技工所可直接匯入
- 可另外下載臨床照片與案件 CSV，供傳檔、Excel 或 Google Sheets 使用
- 診所端與技工所端分別收集手機、相機 A1–D4 共 16 色的 Lab 基礎資料庫
- 基準資料可下載或匯入 JSON，也可匯出 CSV 至 Excel／Google Sheets

## 使用方式

直接開啟 `index.html`，依序匯入「臨床牙＋比色板」及「完成瓷牙＋比色板」兩張照片。臨床照可放 A2、A3 或其他已建立基準的色號；照片放哪個色號，網頁選單就必須選同一色號。

## 案件流程

1. 診所端與技工所端分別上傳手機、相機拍攝的比色板照片，逐一取樣 A1–D4 並建立四套 Lab 資料庫。
2. 診所拍攝臨床牙與比色板並建立案件。
3. 診所將案件 JSON 傳給技工所；需要時可另附臨床照片與 CSV。
4. 技工所匯入案件 JSON，讀取臨床照片與校正後自然牙 Lab，並製作瓷牙。
5. 技工所拍攝完成瓷牙與比色板。
6. `ΔE00 > 3.0` 返回修色；`ΔE00 <= 3.0` 儲存色差並結案。

## 注意

本工具目前為影像比色與病例紀錄原型，不取代標準化多色塊校正、實體比色板或分光測色儀。
