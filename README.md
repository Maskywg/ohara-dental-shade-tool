# 大原牙醫瓷牙比色工具

以瀏覽器本機處理照片的瓷牙比色工具原型。

## 目前功能

- 臨床牙與照片內比色板雙取樣
- 完成瓷牙與照片內比色板雙取樣
- 兩張照片各自獨立校正
- 校正後 CIE Lab 數值
- CIEDE2000 色差比較
- 近似 VITA Classical 色階
- Google Docs 相容報告匯出
- 診所建立案件、技工所讀取、色差判定與結案流程
- 本機 JSON 案件交換格式（預留 Firebase 串接）
- 手機與相機分開建立比色板設備基準

## 使用方式

直接開啟 `index.html`，依序匯入「臨床牙＋比色板」及「完成瓷牙＋同一比色板」兩張照片。每張照片分別移動牙齒與比色板取樣圈。

## 案件流程

1. 分別上傳手機與相機的比色板照片，保存兩套設備基礎 Lab 與校正量。
2. 診所拍攝臨床牙與比色板並建立案件。
3. 技工所讀取目標牙色彩資料並製作瓷牙。
4. 技工所拍攝完成瓷牙與比色板。
5. `ΔE00 > 3.0` 返回修色；`ΔE00 <= 3.0` 儲存色差並結案。

## 注意

本工具目前為影像比色與病例紀錄原型，不取代標準化多色塊校正、實體比色板或分光測色儀。
